Games

Here are some games you can play. You can either play them on your browser (on itch.io or GameJolt) or download them to play offline.

Keep in mind that on GameJolt the games show an ad on start. A small amount of the revenue generated by said ads goes to me. Thank you in advance if you decide to play there! Otherwise, the rest of the options available are completely ad-free.

On Mac OS X, if you can't run the game be sure you have the option to open apps from unidentified developers checked!

On Linux you need some dependencies to be able to run games. Check the readme that comes with each game for more info.

  • Sacrifice

    Sacrifice is one of those games. Challenging platformer where you die a lot. This one is a work-in-progress, and has just a handful of levels.

    Try getting to the end!

    • Download or play on the browser on GameJolt

    • Download v0.4 preview for Mac OS X
    • Download v0.4 preview for Linux

  • Medic Panic!

    You're Dr. Hilton, and your shift on the hospital just got crazy when Mr. Kent, a millionaire philanthropist, announces he wants to come for a visit!

    Fast-paced action arcade game with 14 levels. Heal all the patients on each floor!

    Keep in mind this is a Release Candidate! If you find any bug, or want to give feedback, please fill the attached form!

  • Rescue

    Rescue is a retro-styled arcade platformer about a groundhog that jumps and throws rocks at stuff. 8 levels, 3 difficulties.

  • Speedster

    Speedster is an arcade driving game inspired by Taito's Super Speed Race. Since the original game is rather obscure and it's not well emulated, I decided to make one to play myself. It turned out okay.
